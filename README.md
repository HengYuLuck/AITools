# 🏮 宝宝智能起名神器

> 传承千年命理智慧，AI科学起好名

## 📋 项目概述

这是一个基于传统中国命理学和现代算法的智能宝宝起名工具。通过分析生辰八字、五行缺失，结合丰富的汉字字库，为新生儿推荐寓意美好、五行匹配的名字。

## ✨ 技术特色

### 🎯 真实八字计算
- **儒略日算法**：精确计算日柱，确保八字准确性
- **节气界定**：以立春为界计算年柱，以节气为界计算月柱
- **时辰换算**：准确换算现代时间为传统时辰

### 🔍 专业五行分析
- **元素统计**：统计八字中金木水火土的分布
- **强弱判断**：智能分析五行强弱，找出需要补充的元素
- **用字建议**：根据五行缺失推荐相应属性的汉字

### 📚 丰富字库系统（重要更新）
- **模块化设计**：字库已抽离为独立文件 `nameDatabase.js`
- **基础字库**：包含约80个精选字符，涵盖五行属性
- **扩展字库**：`expandDatabase.js` 提供额外80个字符
- **字库统计**：可通过 `database-stats.html` 查看字库详情
- **易于维护**：支持动态扩展，便于添加更多字符

#### 字库文件结构
```
nameDatabase.js     - 基础字库（80个核心字符）
expandDatabase.js   - 扩展字库（80个附加字符）
database-stats.html - 字库统计查看页面
```

### 🧮 多维度评分算法
- **五行匹配度**（40%）：优先考虑补充所需五行元素
- **笔画数理**（25%）：基于传统姓名学笔画吉凶
- **音韵美感**（20%）：分析声调搭配的和谐度  
- **寓意吉祥**（15%）：考虑字符的文化内涵和美好寓意

### 🎲 智能随机生成
- **Fisher-Yates算法**：确保名字生成的随机性
- **去重机制**：避免重复名字的产生
- **候选池扩大**：生成20个候选名字，筛选出最佳5个

## 🚀 使用方法

1. **打开应用**：在浏览器中打开 `index.html`
2. **填写信息**：
   - 输入宝宝姓氏
   - 选择宝宝性别
   - 设置出生日期和时间
3. **点击起名**：系统自动分析并推荐名字
4. **查看结果**：
   - 详细的八字信息
   - 五行分析报告
   - 5个推荐名字及其详细解释
